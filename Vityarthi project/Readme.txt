## Project Objective
The Health Habit Tracker helps users build and track daily habits related to health and wellness. It allows users to add habits, log daily completion, track streaks of consecutive successful days, and view weekly completion percentages.

# Features
- Add new habits with descriptive goals
- Log daily progress for each habit
- Check current streaks for habits (days in a row completed)
- View completion percentage for the last 7 days
- Save habit data and logs to a file and load them back

# Technologies Used
- Python 3.14
- Built-in   Python Libraries like `datetime` and `Json` used for date handling and data storage.
-Ide- Visual Studio (VS CODE).
